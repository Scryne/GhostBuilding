# Data Sources & Licensing

GhostBuilding aggregates public geospatial information. We rely on the following primary sources:

## Base Providers
1. **OpenStreetMap (OSM)**
   - Used for structural ground-truth and road matrices via Overpass API.
   - **License:** Open Data Commons Open Database License (ODbL).
2. **Google Maps / Earth**
   - Used for primary satellite imagery (high residential resolution).
   - **License:** Proprietary. Usage strictly caching temporal anomalies for fair-use research.
3. **Bing Maps**
   - Secondary satellite imagery provider for validation.
4. **Yandex Maps**
   - High value for verifying Eastern European and Northern Asian topographies.

## Historical Datasets
- **NASA GIBS** (Global Imagery Browse Services) - Public Domain.
- **Sentinel Hub** - Copernicus openly available data.
